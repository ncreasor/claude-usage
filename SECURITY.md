# Security Policy

## Scope

This tool reads your local browser cookie database to authenticate with claude.ai. No credentials are stored or transmitted to any third party — all requests go directly to Anthropic.

If you find a vulnerability (e.g. local privilege escalation, credential leak), please report it privately.

## Reporting

Open a [GitHub issue](https://github.com/ncreasor/claude-usage/issues) marked **[security]** in the title, or email the address on the GitHub profile.

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
